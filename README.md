# AI camera witt Gemini

## setup
1. 仮想環境構築
```
python3 -m venv [envname]
```

2. 仮想環境に入る
```
//On Windows
[envname]\Scripts\activate

//On Unix or MacOS
source [envname]/bin/activate
```

3. 依存関係インストール
```
pip install -r requirements.txt
```

4. 実行
```
python main.py
```

5. 仮想環境から出る
```
deactivate
```
